# Mid-Project: Online Retail Analysis

This repository contains the mid-project analysis for online retail transactions.

## Contents
- `data/raw`: Original dataset
- `data/processed`: Cleaned dataset used for analysis
- `notebooks`: Jupyter notebook with full analysis and visualizations

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
